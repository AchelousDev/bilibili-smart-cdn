# Bilibili Smart CDN

A Tampermonkey userscript that improves Bilibili 4K playback with smart CDN selection and adaptive media prefetching.

## Features

- Smart Bilibili CDN replacement
- Adaptive video and audio prefetching
- Seek-aware media baseline reset
- Automatic cancellation of outdated prefetch tasks
- Reduced unnecessary repeated downloading
- Designed for high-bitrate and 4K playback

## Requirements

- Tampermonkey
- Chromium-based browser such as Chrome or Edge

## Installation

1. Install Tampermonkey.
2. Open `Bilibili-CDN-Smart-Selector.user.js`.
3. Click **Raw**.
4. Tampermonkey should open the installation page automatically.
5. Click **Install**.

## Notes

Adaptive prefetching may increase bandwidth usage, especially when watching 4K videos.
